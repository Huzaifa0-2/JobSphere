# JobSphere 💼

A full-stack job portal (Indeed-like) built with modern technologies.

## 🚀 Tech Stack

* React / Next.js
* Node.js / Express
* MongoDB
* Clerk Auth
* Redux Toolkit
* Tailwind CSS

## 📌 Features

* User authentication
* Job posting
* Apply to jobs
* Save jobs
* Dashboard
* Search & filters
